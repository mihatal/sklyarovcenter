PJM — карта пути клиента продуктового ведения

Универсальный путь клиента для любого направления (похудение · мышцы+ТРТ · сон · антиэйдж · когнитивка). Базовый путь — один; «начинка» меняется по направлению. Цель сессии — пройти путь и осознать процессы, которые надо создать внутри бизнеса, чтобы клиент шёл гладко. Тарифы и сетка — по зафиксированному лендингу (v1.0, 13 мая).

Как работать на сессии

  1. Слева направо по этапам 0→8. Сверху вниз: мир клиента → линия видимости → что готовит бизнес → бизнес.
  2. Значок = статус (знаем→обсудить→решить→решено). Двойной клик по тексту = правка.
  3. «+отв» на карточке = назначить ответственного и срок. «+ карточка» внизу ячейки = добавить пункт вживую. Навёл на карточку → × = удалить.
  4. Живые заметки в каждом этапе + Парковка решений. Внизу — сводка «кто что забрал».
  5. Сначала открой «Сценарий сессии» ниже. Всё сохраняется автоматически. «Команда» — настроить участников.

Контекст

Клиент: 35–50, состоятельный. Приходит с болью (вес/энергия/гормоны/сон), слова «долголетие» в запросе нет.

Модель: членство/сопровождение. Продаём не препарат, а инфраструктуру жизни под ключ.

Воронка: вход (анализ крови/чекап) → презентация плана → программа (2 тарифа) → ретейнер.

CRM = [впиши CRM клиники] — единое место, клиент закреплён за сотрудником. Старт: своя база (MVP).
